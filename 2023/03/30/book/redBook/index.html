3.5 操作符

一元操作符

  1. 递增和递减操作符
// 前置 递增 递减
var age = 29;
var anotherAge = --age + 2;
alert(age); // 输出 28
alert(anotherAge); // 输出 30
// 后置 递增 递减
var num1 = 2;
var num2 = 20;
var num3 = num1-- + num2; // 等于 22
var num4 = num1 + num2; // 等于 21

这里在计算 num3 时使用了 num1 的原始值（2）完成了加计算，而 num4 则使用了递减后的值 （1）

let b = 1;
let a = b++ + 10;
// 可以理解为
// a = b + 10;
// b = b + 1
  1. 一元加和减操作符
    一元加操作符以一个加号（+）表示，放在数值前面，对数值不会产生任何影响
